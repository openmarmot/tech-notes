# -- EUGR docker setup ---

This is a spark specific repo for vLLM 

Setup 

1. optionally follow the networking guide to connect your spark cluster 
https://github.com/eugr/spark-vllm-docker/blob/main/docs/NETWORKING.md

2. review recipes 
https://github.com/eugr/spark-vllm-docker/tree/main/recipes

3. optionally setup/discover cluster 
./run-recipe.sh --discover
./run-recipe.sh --show-env

4. run a recipe (auto download and setup)
./run-recipe.sh qwen3.5-122b-fp8 --setup
